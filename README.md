## ✨ Project Spotlight

This repository is my **personal playground** for **Applications Development & Emerging Technologies (CCC311-18)**.  

Inside, you'll find **6 progressive Bootstrap activities** — from basic layouts to more interactive components — all wrapped in a clean, modern **responsive dashboard** that showcases everything beautifully.

Whether you're a classmate, instructor, or fellow learner, explore how Bootstrap 5 brings ideas to life with grids, cards, navbars, forms, and more!

## 🚀 Quick Start

1. **Clone the repo**
   ```bash
   git clone https://github.com/JLNerecina/AppDev_Bootstrap_Activities.git
   cd AppDev_Bootstrap_Activities
   ```

2. **Open the magic**
   - Simply double-click `index.html` or
   - Use Live Server (VS Code) for the best experience

3. **For Activity 1 (PHP)**
   ```bash
   php -S localhost:8000
   ```
   Then visit `http://localhost:8000/Activity1`
